# 🎨 Dribbble Clone

A fully responsive Dribbble-inspired landing page built using pure HTML5 and CSS3. This project recreates the modern UI of Dribbble with responsive layouts, interactive design cards, smooth hover effects, animated sections, and mobile-friendly navigation.

---

## Features

- Responsive Navbar with Hamburger Menu
- Mobile First Design
- Hero Section with Animated Heading & Search Bar
- Trending Search Tags
- Category Filter Navigation
- Interactive Design Cards
- Image & Video Card Support
- Hover Overlay Effects
- Designer Profiles & Statistics
- Infinite Scrolling Logo Marquee
- Smooth Button Animations
- Back To Top Button
- Modern Footer Layout
- Fully Responsive Across Devices

---

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- CSS Animations
- Remix Icons

---

## Responsive Breakpoints

| Device | Width |
|----------|----------|
| Mobile | < 576px |
| Tablet | ≥ 576px |
| Large Tablet | ≥ 768px |
| Desktop | ≥ 992px |
| Large Desktop | ≥ 1200px |
